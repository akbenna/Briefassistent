AVG Verantwoording — BriefAssistent v4

Dit document beschrijft hoe BriefAssistent omgaat met medische patiëntgegevens conform de Algemene Verordening Gegevensbescherming (AVG/GDPR).

1. Gegevensverwerking

AspectBeschrijving
VerwerkingsverantwoordelijkeDe huisarts die de extensie gebruikt
VerwerkerAnthropic (Claude API) — verwerkt geanonimiseerde tekst
GrondslagGerechtvaardigd belang (efficiënte briefschrijving) + toestemming arts
DoelGenereren van medische informatiebrieven

2. Privacy-maatregelen

MaatregelStatusToelichting
Naam → initialen ✓ Actief Volledige naam wordt nooit naar de API gestuurd. Alleen initialen.
BSN-filtering ✓ Actief 9-cijferige nummers worden vervangen door [BSN]
Datum-filtering ✓ Actief Geboortedatums en andere datums worden vervangen door [DATUM]
Contactgegevens ✓ Actief Telefoon, e-mail, postcode gefilterd
Sectie-controle ✓ Actief Arts bepaalt per sectie welke data wordt verstuurd
Preview ✓ Actief Arts kan exact zien wat naar de API gaat vóór verzending
Lokale opslag ✓ Actief Alleen API-sleutel wordt lokaal opgeslagen. Geen patiëntdata.
Sessie-gebonden ✓ Actief Alle patiëntdata verdwijnt bij sluiten van de popup

3. Gegevensstroom

Bricks HIS / Screenshot / PDF → Extensie (lokaal in browser) → Privacy-filter (naam, BSN, datums, contactgegevens verwijderd) → Arts controleert via preview → Arts klikt "Genereer" → Geanonimiseerde tekst → Anthropic Claude API (HTTPS) → Gegenereerde brief terug → Weergave in extensie

4. Wat wordt NIET verstuurd

✗Volledige patiëntnaam
✗BSN-nummer
✗Geboortedatum
✗Adres of postcode
✗Telefoonnummer of e-mail
✗Secties die de arts heeft uitgeschakeld

5. Aanbevelingen

  • Controleer altijd de preview vóór het genereren
  • Schakel secties uit die niet relevant zijn voor de brief
  • Controleer de gegenereerde brief op juistheid vóór gebruik
  • Gebruik de extensie alleen op beveiligde werkstations
  • Deel uw API-sleutel niet met derden
BriefAssistent v4 — Ontwikkeld als hulpmiddel voor huisartsen. De arts blijft te allen tijde verantwoordelijk voor de inhoud van verstuurde brieven.
